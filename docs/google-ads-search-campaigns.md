Manual steps in Google Ads:
1. Create the campaigns and ad groups.
2. Add the keywords from the CSV file.
3. Add the shared negative keyword list.
4. Create one responsive search ad per ad group using the copy above.
5. Link GA4 and import `generate_lead` as a primary conversion.
6. Add sitelinks for `Work`, `Services`, `Contact`, and `Talents`.
